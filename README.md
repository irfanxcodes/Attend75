# Attend75 Monorepo

This repository is organized as a monorepo-style project.

## Structure

- frontend/: React + Vite frontend application
- backend/: Reserved for backend service (currently empty)

## Run Frontend

From repository root:

```bash
npm run dev
```

From frontend folder:

```bash
cd frontend
npm run dev
```

## Build Frontend

From repository root:

```bash
npm run build
```
